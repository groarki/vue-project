# IT Travels

**IT Travels** is a learning project created during the **Vue.js** course at **GoIT**

The project follows the design [mockup](https://www.figma.com/design/iKGFonhDEQiTWjSf4pa3P6/IT-traveler?node-id=0-1&p=f&t=4McNTyIpR9PLr9Pb-0)

## Project Description

IT Travels allows users to easily mark their favorite places on a map and save them in one place.  
Users can quickly access their markers, add new ones, edit, or delete them.

Key features:

- Minimalistic and pleasant design.
- User authentication.
- Create custom markers on the map.
- Full **CRUD** implementation.

## Technologies

- **Vue.js**
- **Vue Router**
- **Axios**
- **TailwindCSS**

## Installation and Running

1. Clone the repository:

```bash
git clone git@github.com:groarki/vue-project.git
```

2. Install all dependencies by running

```bash
npm install
```

3. Run the project

```bash
npm run dev
```

Feel free to clone the repo and explore IT Travels locally or visit the deployed version!
